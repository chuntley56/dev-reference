# CSS Notes

## Combinators
### General combinator - any descendant
  Code: `space`<br />
  Example: `.parent .child`<br />
  Notes: `works for any child regardless of how deep`

### Child combinator - direct descendant
  Code: `>`<br />
  Example: `.parent > .child`<br />
  Notes: `only works on direct descendants`
